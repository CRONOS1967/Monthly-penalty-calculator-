## ✨ Features

- **Compounding Penalties**: Each month's penalty is calculated on base payment plus all previous unpaid amounts
- **Custom Rate Tables**: Configurable penalty rates by month (default table for 36 months included)
- **High Precision**: Decimal-based rounding with configurable precision
- **Batch Processing**: Handle multiple calculations efficiently
- **Multiple Export Formats**: JSON, CSV, pandas DataFrame, dictionaries
- **Type Hints**: Full typing support for better IDE integration
- **Comprehensive Documentation**: Detailed docstrings and examples
- **Production Ready**: Error handling, validation, and testing support
